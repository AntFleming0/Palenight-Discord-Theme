üì• Installation

Quick Install

  1. Install BetterDiscord if you haven't already
  2. Open Discord Settings ‚Üí Themes ‚Üí Open Theme Folder
  3. Create a new file: VSCodePalenight.theme.css
  4. Paste this code into the file:
/** * @name VSCode Palenight * @description VSCode-inspired Palenight theme * @author AntFleming0 * @version 1.0.0 */ @import url('https://antfleming0.github.io/BetterDiscordTheme/theme.css');
  1. Save the file and enable the theme in Discord Settings

Theme Variants

To use different variants, change the import URL:

  • Palenight (Default): theme.css
  • Dark (Higher Contrast): theme-dark.css
  • Light: theme-light.css

üîß Customization

Customize the theme by adding CSS variables after the import:

@import url('https://antfleming0.github.io/BetterDiscordTheme/theme.css'); :root { /* Change accent color */ --accent-primary: #FF6B9D; /* Add background image */ --custom-background-image: url('YOUR_IMAGE_URL'); --custom-background-blur: 5px; /* Change font */ --font-primary: 'Fira Code', monospace; }

See the full customization guide for all available options.
