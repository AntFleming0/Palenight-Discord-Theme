## How to Update

When a new version is released:

1. **Automatic (Recommended):** If you're using the GitHub Pages @import method, your theme updates automatically
2. **Manual:** Download the latest release from GitHub and replace your theme file
3. **Git:** Pull the latest changes from the repository

Always check this changelog before updating to see what's new!
